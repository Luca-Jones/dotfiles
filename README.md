# Dotfiles

## Requirements

- git
- stow

## Installation

```
git clone https://github.com/Luca-Jones/dotfiles
cd dotfiles
stow .
```

### Install yay

```
git clone https://aur.archlinux.org/yay.git
cd yay
mkpkg -si
```
